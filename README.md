# Lorien's Library

Independent AI safety research and applied systems by **Angela Reinhold**.
ORCID: [0009-0005-5803-8401](https://orcid.org/0009-0005-5803-8401)

## Projects

- **[CAMA](https://github.com/LoriensLibrary/cama)** — Circular Associative Memory Architecture. A provenance-aware, three-layer persistent memory system for human–AI interaction, with explicit write discipline separating user-authored teachings from assistant-generated inferences. ~270 KB of Python, 34 MCP tools, real operational scale (52,900+ memories across 825 conversations). MIT licensed.

- **[Telos · for Kalos](https://github.com/LoriensLibrary/Telos_kalos)** — Working React 19 + TypeScript prototype of an AI continuity layer between DEXA scans. Built independently for the Kalos Health Software Engineer role, mapping directly to their work-stream #3. Includes a CAMA Proof Layer demonstrating end-to-end provenance trace: coach insight → derived patterns → source memory records. Live at [telos-kalos.vercel.app](https://telos-kalos.vercel.app). 27 tests across 4 suites, GitHub Actions CI, Vercel deploy.

- **[Project Companion](https://github.com/LoriensLibrary/Project-Companion)** — Persistent-memory AI learning companion for K–12 education. Three-sided platform (student, teacher, parent) built on CAMA. Designed as the applied deployment of the published safety architecture, with COPPA compliance, provenance-aware memory, and teacher-controlled curriculum injection.
